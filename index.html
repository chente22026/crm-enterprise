CRM Enterprise

Scaffold inicial del proyecto CRM Enterprise — monorepo (frontend, backend, infra).

Estado

  • Archivo de inicio: index.html (este archivo)
  • Objetivo: primer commit para permitir creación de ramas y PRs.

Próximos pasos recomendados

  1. Hacer commit y push de este archivo a la rama main.
  2. Crear la rama scaffold/initial y pushear los archivos del scaffold (o ejecutar el script local que te pasé).
  3. Abrir PR: scaffold/initial → main.

Comandos útiles (en tu terminal)

# Crear el index.html con contenido arriba, luego:
git add index.html
git commit -m "chore: add landing index.html (initial commit)"
git branch -M main

# Si aún no añadiste el remoto:
git remote add origin https://github.com/chente22026/crm-enterprise.git
# o con SSH:
# git remote add origin git@github.com:chente22026/crm-enterprise.git

git push -u origin main

# Crear y pushear la rama scaffold/initial
git checkout -b scaffold/initial
git push -u origin scaffold/initial

# (Opcional, si tienes gh CLI y estás autenticado)
gh pr create --title "Initial scaffold" --body "Initial scaffold branch" --base main --head scaffold/initial
Si quieres, después de pushear la rama scaffold/initial me pasas el enlace al PR y yo reviso el scaffold, creo issues/epics y añado sugerencias.
